# SkillLedger Landing Page

**SkillLedger** is a skills data infrastructure platform designed to make learners' abilities **visible, verifiable, and bankable**. This repository hosts the **landing page**, showcasing the concept, features, and pilot program — the system itself will be developed separately.

---

## 🚀 Overview

SkillLedger is the **trust layer** between:

- **TVET institutions** – capturing real learning events.
- **Employers** – assessing verified skill profiles.
- **Lenders** – unlocking data-driven financing for skills development.

Key ideas:

- **Verified Skills Scores**: Convert real learning events into trusted scores.
- **Behavioral scoring**: Consistency, retries, and performance tracked.
- **Tamper-proof data**: Audit logs ensure trust.
- **Portable Skills Passport**: A digital profile you can take anywhere.
