How It Works

From learning event to verified score

A four-step trust pipeline that turns everyday learning into economic proof.

Step 01

Data Ingestion

Partner TVET or bootcamp sends learning events via POST /events. Works with Excel, LMS, or direct entry. Zero new tools for teachers.

Step 02

Async Processing

Events are validated, deduplicated (idempotency), and queued via Redis/Celery. Tamper-proof audit chain is written at every step.

Step 03

Scoring Engine

CBET-aligned weighted algorithm runs: Performance 45% + Behavioral 30% + Validation 15% + Progress 10% = Skills Score.

Step 04

Verified Profile

Employer queries GET /profiles/{id}. Lender receives Trust Score API signal. Student gets portable Skills Passport.

What Is SkillLedger?

Infrastructure, not an app

Think Plaid for education data — a secure, standardized pipeline that makes skills data trustworthy and actionable.

Data Infrastructure

API-first ingestion layer captures learning events — practical tasks, assessments, project results — directly from institutions with zero friction.

Behavioral Scoring

We track consistency, retries, and improvement curves — the hidden signals that predict real-world performance better than any single exam.

Tamper-Proof Trust Layer

Hash-linked audit logs + idempotent ingestion delivers fintech-grade security. Every score change is immutably recorded and verifiable.

Portable Skills Passport

A verifiable digital identity students carry beyond their institution — unlocking jobs and "Study Now, Pay Later" education financing.
